skills (Selected Relevant Courses and Concepts).

Introduction to Object-Oriented Programming [Fall 2016] -An introduction to procedural and object-oriented programming methodology. -Topics include program structure, conditional and iterative programming, procedures, arrays and records, object classes, encapsulation, information hiding, inheritance, polymorphism, file I/O, and exceptions.
Data Structures [Spring 2017] -An extension of programming methodology to data storage and manipulation on complex data sets. -Topics include programming and applications of data structures, Recursive programming, fundamental sorting and searching algorithms.
Computer Science III [Fall 2017] -Development of the basic concepts and techniques learned into practical programming skills that include a systematic approach to program design, coding, testing, and debugging. -Use of programming environments and tools to aid in the software development process.
Systems Fundamentals I [Spring 2018] -Introduce assembly language programming and essential concepts of computer organization and architecture. -The focus of this course is on the computer organization of a computer system, including the processor architecture and the memory system.
Principles of Database Systems [Spring 2018] -The design of database management systems to obtain consistency, integrity, and availability of data. -Semester project that includes the design and implementation of a database system.
Principles of Programming Languages [Fall 2018] -Presents examples of important programming languages and paradigms such as LISP, ALGOL, ADA, ML, Prolog, and C++. Students write sample programs in some of the languages studied. -The languages are used to illustrate programming language constructs. operations, data control, storage management, parameter passing, and operating environment.
Fundamentals of Computer Vision [Spring 2019] -Introduces fundamental concepts, algorithms, and techniques in visual information processing. -Covers image formation, binary image processing, image features, model fitting, optics, illumination, texture, motion, segmentation, and object recognition.
Research in Computer Science [Fall 2018] -An independent research project with faculty supervision.
